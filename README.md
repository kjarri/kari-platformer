# Hoppa

A retro-style platformer game built with vanilla JavaScript and HTML5 Canvas.

## How to Play

- **A / D** - Move left/right
- **W / Space** - Jump
- **Arrow Keys** - Shoot in different directions

## Game Features

- 5 levels with increasing difficulty
- Side-scrolling platformer action
- Collect items and defeat enemies
- Debug mode for development

## Running the Game

Simply open `index.html` in any modern web browser.

```bash
# Or serve it locally
python3 -m http.server 8000
```

Then visit http://localhost:8000

## Tech Stack

- HTML5 Canvas
- Vanilla JavaScript
- No dependencies
